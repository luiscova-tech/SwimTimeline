# Narwhal Invite Calendar Output

This folder will hold generated calendar files for the 2026 MAC Narwhal Invite.

Current status: generated from verified individual events.

Outputs:

- `mila-narwhal-invite-daily.ics`: preferred Google Calendar import file with one calendar event per meet day.
- `mila-narwhal-invite.ics`: detailed calendar import file with 7 verified swim events.
- Event preview table: `../extracted/mila-timeline.md`

The `.ics` file includes verified morning/timed-final events only. Possible finals are included in event descriptions, but not added as separate calendar events because qualification is not yet known.
